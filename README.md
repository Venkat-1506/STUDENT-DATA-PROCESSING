# 📊 Student Data Processing System

A Python-based project that reads student data from a CSV file, processes it, and stores the results in JSON format.

---

## 🚀 Features
- Reads student data from CSV file
- Calculates total and average marks
- Assigns grades (A, B, C, F)
- Handles invalid or missing data
- Converts processed data into JSON format

---

## 🛠️ Technologies Used
- Python
- CSV Module
- JSON Module
- File Handling
